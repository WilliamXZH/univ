2.2.3 W3C发布的标准

网页主要由三部分组成：
结构(structure)，表现(Presentation)，行为(Behavior)

对应的网站标准也分为三部分：
结构化标准语言，主要包括HTML,XHTML,XML
表现标准语言，主要包括CSS
行为标准，主要包括对象模型(如W3C DOM),ECMAScript等

2.2.4 Web标准的优势

对网站浏览者的好处

对网站设计者的好处：
更少的代码，容易维护
Web标准强制用户进行错误校验
节约带宽成本
更容易被搜寻引擎搜索到
改版方便，不需要变动页面内容
提供打印版本而不需要复制内容
提高网站易用性

2.3.2如何改善现有网站

初级改善:
*为页面添加正确的DOCTYPE
*设定一个命名空间
<html XMLns="http://www.w3.org/1999.xhtml">
*声明编码语言
<meta http-equiv="Content - Type" content="text/html;charset=GB2312" />
*用小写字母书写所有的标签
*为图片添加alt属性
<img src="logo.gif" alt="东方公司标志，首页">
*给所有属性值加引号
*关闭所有的标签

中级改善：
*用CSS定义元素外观
*用结构化元素代替无意义的垃圾代码
*给每个表格和表单加上id
